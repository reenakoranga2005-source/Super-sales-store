# Super sales store - Power BI Project
## Project Title
**Super Sales Store Business Insight Dashboard**

---

## Project Description/Purpose
This project is an interactive Power BI dashboard designed to analyze business sales performance.
The Objective is to the success of a business by utilizing data analysis techniques.

The dashboard focuses on:
- time series analysis
- Sales performance
- Profit analysis
- Regional trends
- Category-wise insights
- Monthly growth patterns

  ---

  ## Tech Stack
- 📊 Power BI Desktop - Main data Visualization platform used for report creation.
- 📂 Excel / CSV Dataset - Contains information about super sales store.
- 🔄 Power Query - Data transformation and cleaning layer for reshaping and preparing the data.
- 📂 File Format - .pbix for development and.png for dashboard previews.
- 🧮 DAX (Data Analysis Expressions) - Uesd for calculated measures,dynamic visual and conditional logic.

## ✨ Features / Highlight

🧩 Business Problem

The retail supermarket industry generates significant revenue, but decision-makers often struggle to quickly interpret large volumes of sales data.

Key questions such as: Which region generates the highest revenue? Which product categories are the most profitable? Are there loss-making segments affecting overall profit? What are the monthly sales trends and seasonal patterns?What can we expect sales to look like in the next few weeks?-… are difficult to answer efficiently using raw spreadsheets alone.

🎯 Goal of the Dashboard

To deliver an interactive business intelligence dashboard that :
Enables stakeholders to monitor overall sales performance in real time.
Identifies high-performing and underperforming regions.
Analyzes category-wise and sub-category-wise profitability.
Tracks monthly sales trends and seasonality.
Provides short-term sales forecasting (15-day prediction).
Supports data-driven decision-making for growth and efficiency.
The dashboard transforms raw transactional data into clear, actionable insights.

## 📊 Walkthrough of Key Visuals

### 🔹 KPI Cards
- Total Sales  
- Total Profit  
- Total Quantity  
- Average Delivery Time  

### 🔹 Region-wise Sales Analysis
Compare sales performance across different regions to identify top-performing areas.

### 🔹 Category & Sub-Category Breakdown (Clustered Bar Chart)
Provides a detailed comparison of product category performance.

### 🔹 Monthly Sales Trend Analysis (Area Chart)
Helps identify seasonal patterns and overall sales growth trends.

### 🔹 Profit Distribution (Donut Chart)
Analyzes profitability contribution across different segments.

### 🔹 Interactive Slicers & Filters
Allows users to dynamically filter data for deeper analysis.

### 🔹 Drill-down Capability
Enables exploration from high-level summaries to detailed transactional data.
